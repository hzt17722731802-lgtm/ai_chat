## AI 对话接口

### 基础信息
- **接口前缀**: `/ai`
- **流式响应**: 是（text/event-stream）
- **字符编码**: UTF-8

### 1. AI 智能对话

#### 请求
- **URL**: `/ai/chat`
- **Method**: `GET`
- **参数**:
  - `prompt`: 用户输入的问题（必填）
  - `chatId`: 会话ID（可选，不传则自动生成UUID）

#### 请求示例
```
GET /ai/chat?prompt=你好&chatId=abc123
GET /ai/chat?prompt=糖尿病有什么症状
```

#### 响应
- **Content-Type**: `text/html;charset=utf-8`
- **响应类型**: 流式响应（Server-Sent Events）

#### 响应示例
```
你好！有什么可以帮助你的吗？
```

#### 前端调用示例

```javascript
// 使用 fetch 进行流式对话
const chat = async (prompt, chatId = null) => {
  const url = new URL(`${BASE_URL}/ai/chat`);
  url.searchParams.append('prompt', prompt);
  if (chatId) {
    url.searchParams.append('chatId', chatId);
  }

  try {
    const response = await fetch(url);

    if (!response.ok) {
      throw new Error(`HTTP error! status: ${response.status}`);
    }

    const reader = response.body.getReader();
    const decoder = new TextDecoder('utf-8');
    let fullResponse = '';

    while (true) {
      const { done, value } = await reader.read();
      if (done) break;
      const chunk = decoder.decode(value, { stream: true });
      fullResponse += chunk;
      console.log('Received:', chunk);
    }

    return fullResponse;
  } catch (error) {
    console.error('Chat error:', error);
    throw error;
  }
};

// 使用示例
chat('糖尿病有什么症状').then(response => {
  console.log('Full response:', response);
});

// 带有 chatId 的示例（继续对话）
chat('糖尿病有什么症状', 'session-123').then(response1 => {
  console.log('First response:', response1);

  chat('如何预防？', 'session-123').then(response2 => {
    console.log('Second response:', response2);
  });
});
```

## 对话历史接口

### 基础信息
- **接口前缀**: `/ai/history`

### 2. 获取会话ID列表

#### 请求
- **URL**: `/ai/history/{type}`
- **Method**: `GET`
- **路径参数**:
  - `type`: 会话类型（如 "chat"）

#### 请求示例
```
GET /ai/history/chat
```

#### 响应
```json
{
  "code": 200,
  "message": "操作成功",
  "data": [
    "abc123",
    "def456",
    "ghi789"
  ]
}
```

#### 前端调用示例
```javascript
const getChatIds = async (type = 'chat') => {
  const response = await fetch(`${BASE_URL}/ai/history/${type}`, {
    method: 'GET',
    headers: {
      'Authorization': 'Bearer your_token'
    }
  });

  const result = await response.json();
  return result;
};

// 调用示例
getChatIds('chat').then(chatIds => {
  console.log('Chat IDs:', chatIds);
});
```

### 3. 获取对话历史消息

#### 请求
- **URL**: `/ai/history/{type}/{chatId}`
- **Method**: `GET`
- **路径参数**:
  - `type`: 会话类型（如 "chat"）
  - `chatId`: 会话ID

#### 请求示例
```
GET /ai/history/chat/abc123
```

#### 响应

#### MessageVO 字段说明
| 字段名 | 类型 | 说明 |
|--------|------|------|
| role | String | 消息角色：user（用户）、assistant（AI助手） |
| content | String | 消息内容 |
| createTime | LocalDateTime | 创建时间（格式：yyyy-MM-ddTHH:mm:ss） |

#### 前端调用示例
```javascript
const getChatHistory = async (type = 'chat', chatId) => {
  const response = await fetch(`${BASE_URL}/ai/history/${type}/${chatId}`, {
    method: 'GET',
    headers: {
      'Authorization': 'Bearer your_token'
    }
  });

  const result = await response.json();
  return result;
};

// 调用示例
getChatHistory('chat', 'abc123').then(messages => {
  console.log('Chat history:', messages);
  messages.forEach(msg => {
    console.log(`${msg.role}: ${msg.content}`);
  });
});
```

## 实体说明

### MessageVO 字段
| 字段名 | 类型 | 说明 |
|--------|------|------|
| role | String | 消息角色：user（用户）、assistant（AI助手）、unknown（未知） |
| content | String | 消息内容 |
| createTime | LocalDateTime | 创建时间（格式：yyyy-MM-ddTHH:mm:ss） |

### ChatHistory 字段（数据库实体）
| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | String | 记录ID（UUID） |
| chatId | String | 会话ID |
| role | String | 消息角色 |
| content | String | 消息内容 |
| createTime | LocalDateTime | 创建时间 |

## 前端集成示例

### Vue 3 组件示例
```vue
<template>
  <div>
    <h1>AI 智能助手</h1>

    <div class="chat-container">
      <div class="message-list">
        <div
          v-for="(msg, index) in messages"
          :key="index"
          :class="['message', msg.role]"
        >
          <div class="role-label">{{ msg.role === 'user' ? '我' : 'AI' }}</div>
          <div class="content">{{ msg.content }}</div>
          <div class="time">{{ formatTime(msg.createTime) }}</div>
        </div>
      </div>

      <div class="input-area">
        <textarea
          v-model="inputText"
          placeholder="请输入您的问题..."
          @keydown.enter.exact.prevent="sendMessage"
        ></textarea>
        <button @click="sendMessage" :disabled="loading">
          {{ loading ? '思考中...' : '发送' }}
        </button>
      </div>
    </div>
  </div>
</template>

<script setup>
import { ref } from 'vue';

const inputText = ref('');
const messages = ref([]);
const loading = ref(false);
const currentChatId = ref(null);

const sendMessage = async () => {
  if (!inputText.value.trim() || loading.value) return;

  const userMessage = {
    role: 'user',
    content: inputText.value,
    createTime: new Date().toISOString()
  };

  messages.value.push(userMessage);
  const prompt = inputText.value;
  inputText.value = '';
  loading.value = true;

  try {
    const url = new URL(`${BASE_URL}/ai/chat`);
    url.searchParams.append('prompt', prompt);
    if (currentChatId.value) {
      url.searchParams.append('chatId', currentChatId.value);
    }

    const response = await fetch(url);

    if (!response.ok) {
      throw new Error('请求失败');
    }

    const reader = response.body.getReader();
    const decoder = new TextDecoder('utf-8');
    let fullResponse = '';

    while (true) {
      const { done, value } = await reader.read();
      if (done) break;
      const chunk = decoder.decode(value, { stream: true });
      fullResponse += chunk;
    }

    if (!currentChatId.value) {
      currentChatId.value = prompt;
    }

    const aiMessage = {
      role: 'assistant',
      content: fullResponse,
      createTime: new Date().toISOString()
    };

    messages.value.push(aiMessage);
  } catch (error) {
    console.error('Chat error:', error);
    alert('发送失败，请重试');
  } finally {
    loading.value = false;
  }
};

const formatTime = (timeStr) => {
  if (!timeStr) return '';
  const date = new Date(timeStr);
  return date.toLocaleTimeString('zh-CN', { hour: '2-digit', minute: '2-digit' });
};
</script>
```

### React 组件示例
```javascript
import React, { useState } from 'react';
import axios from 'axios';

const BASE_URL = 'http://localhost:8080';

const ChatComponent = () => {
  const [inputText, setInputText] = useState('');
  const [messages, setMessages] = useState([]);
  const [loading, setLoading] = useState(false);
  const [chatId, setChatId] = useState(null);

  const sendMessage = async () => {
    if (!inputText.trim() || loading) return;

    const userMessage = {
      role: 'user',
      content: inputText,
      createTime: new Date().toISOString()
    };

    setMessages(prev => [...prev, userMessage]);
    const prompt = inputText;
    setInputText('');
    setLoading(true);

    try {
      const url = new URL(`${BASE_URL}/ai/chat`);
      url.searchParams.append('prompt', prompt);
      if (chatId) {
        url.searchParams.append('chatId', chatId);
      }

      const response = await axios.get(url.toString(), {
        headers: {
          'Accept': 'text/html;charset=utf-8'
        }
      });

      if (!chatId) {
        setChatId(prompt);
      }

      const aiMessage = {
        role: 'assistant',
        content: response.data,
        createTime: new Date().toISOString()
      };

      setMessages(prev => [...prev, aiMessage]);
    } catch (error) {
      console.error('Chat error:', error);
      alert('发送失败，请重试');
    } finally {
      setLoading(false);
    }
  };

  const handleKeyPress = (e) => {
    if (e.key === 'Enter' && !e.shiftKey) {
      e.preventDefault();
      sendMessage();
    }
  };

  const formatTime = (timeStr) => {
    if (!timeStr) return '';
    const date = new Date(timeStr);
    return date.toLocaleTimeString('zh-CN', { hour: '2-digit', minute: '2-digit' });
  };

  return (
    <div style={{ maxWidth: '800px', margin: '0 auto', padding: '20px' }}>
      <h1>AI 智能助手</h1>

      <div style={{ maxHeight: '500px', overflowY: 'auto', marginBottom: '20px' }}>
        {messages.map((msg, index) => (
          <div
            key={index}
            style={{
              marginBottom: '15px',
              padding: '10px',
              borderRadius: '8px',
              backgroundColor: msg.role === 'user' ? '#e3f2fd' : '#f5f5f5',
              marginLeft: msg.role === 'user' ? '50px' : '0',
              marginRight: msg.role === 'user' ? '0' : '50px'
            }}
          >
            <div style={{ fontWeight: 'bold', marginBottom: '5px' }}>
              {msg.role === 'user' ? '我' : 'AI'}
            </div>
            <div style={{ whiteSpace: 'pre-wrap' }}>{msg.content}</div>
            <div style={{ fontSize: '12px', color: '#999', marginTop: '5px' }}>
              {formatTime(msg.createTime)}
            </div>
          </div>
        ))}
      </div>

      <div style={{ display: 'flex', gap: '10px' }}>
        <textarea
          value={inputText}
          onChange={(e) => setInputText(e.target.value)}
          onKeyPress={handleKeyPress}
          placeholder="请输入您的问题..."
          style={{
            flex: 1,
            height: '60px',
            padding: '10px',
            border: '1px solid #ddd',
            borderRadius: '4px',
            resize: 'none'
          }}
        />
        <button
          onClick={sendMessage}
          disabled={loading}
          style={{
            padding: '10px 20px',
            backgroundColor: loading ? '#ccc' : '#1976d2',
            color: 'white',
            border: 'none',
            borderRadius: '4px',
            cursor: loading ? 'not-allowed' : 'pointer'
          }}
        >
          {loading ? '思考中...' : '发送'}
        </button>
      </div>
    </div>
  );
};

export default ChatComponent;
```

## 注意事项

1. **会话管理**: 首次对话时不传 chatId，系统会自动生成 UUID 作为会话ID，后续对话请携带该 chatId 以保持上下文连贯

2. **流式响应**: `/ai/chat` 接口返回流式数据（text/event-stream），前端需要使用 `fetch` + `ReadableStream` 或 `EventSource` 来处理

3. **字符编码**: 接口使用 UTF-8 编码，确保前端正确处理中文

4. **历史记录存储**: 对话消息同时存储在 MySQL（持久化）和 Redis（ChatMemory，用于上下文理解）中

5. **错误处理**: 建议前端对网络错误和业务错误进行统一处理

6. **性能优化**: 避免发送过于频繁的请求，AI 模型需要一定时间处理

7. **跨域**: 对话接口默认支持跨域访问
